Seawards:

A responsive 3D arcade style boat platformer designed to test and hone core movement mechanics and dynamic level progression within the Godot Engine.
 

Gameplay and Objective:

In Seaward you are in command of a fast craft on challenging waterways. Your main task is to travel around the map and pass through all the gates marked on it. If you clear all the gates , you unlock the last gate and can proceed to the next level .

Control Strategies:
W / Up Arrow : Move Forward
S/ Down Arrow : Go Backwards
A/ Left Arrow: Turn Left 
D / Right Arrow Turn Right

Characteristics:

Arcade Boat Physics Responsive, snappy and instant handling for precision maneuvering without frustrating drift.
Dynamic Gate System: Built on an auto-scaling group-tagging architecture (gates and final_gate) that can handle any custom level layout. Unique Node Identification: Accurate collision detection with robust player tracking, no longer dependent on rigid hardcoded pathing.
Clean UI: Dynamic on-screen counter to track remaining objectives in real-time.

Built With :

Engine: Godot Engine 
Language: GD Script

Background & Inspiration:

This project was designed as a practical hands-on exercise to learn how to scale 3D environments, work with signals and isolate player physics in Godot. The whole development focus was on low-latency control feedback, minimal code overhead and a polished core gameplay loop.

Future Improvements:
Audio Design: Design immersive ambient water loops, motor acceleration sound effects, and satisfying gate-collection cues.
Multi-Tiered Environments: Build multi-tiered environments through procedural placement of assets.
Time Attack Mode: Add an active timer countdown and local high score saving system to increase replay ability.
UI/UX Overhaul: Main menu and settings for paused state and animated HUD transitions.
